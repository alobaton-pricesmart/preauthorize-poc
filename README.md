# preauthorize-poc

Custom PreAuthorize flow with Spring Security.

## How to start?

In order to start the POC use:

```bash
$ git clone https://github.com/alobaton/preauthorize-poc.git

$ cd preauthorize-poc
$ mvn clean install

$ mvn spring-boot:run
```
